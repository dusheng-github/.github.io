# 文本溢出

示例： 单行文本溢出 & 多行文本溢出
飞流直下三千尺， 疑是银河落九天

日照香炉生紫烟， 遥看瀑布挂前川。 飞流直下三千尺， 疑是银河落九天。 
/* 单行文本溢出 */
overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;

/* 多行文本溢出 */
display: -webkit-box;
-webkit-box-orient: vertical;
-webkit-line-clamp: 3;
overflow: hidden;
text-overflow: ellipsis;

# 水平垂直居中(推荐 3 种)

<div class="father">
  <div class="son"></div>
</div>

/* 1. margin auto */
.father {
  position: relative;
}
.son {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  margin: auto;
}

/* 2. absolute + transform */
.father {
  position: relative;
}
.son {
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}

/* 3. flex */
.father {
  display: flex;
  justify-content: center;
  align-items: center;
}
.son {
}
